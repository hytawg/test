# ひらがな練習アプリ 要件定義書

## 1. プロジェクト概要

### 目的
未就学児（3〜6歳）がひらがな46文字を楽しみながら自然に覚えられるWebアプリを開発する。
子どもが一人でも操作できるシンプルさと、続けたくなるゲーム性を両立させる。

### ターゲットユーザー
- **主対象**: 3〜6歳の未就学児
- **副対象**: 一緒に使う保護者（進捗確認・設定変更）

---

## 2. ユーザーストーリー

| # | ユーザー | やりたいこと | 理由 |
|---|---------|-------------|------|
| 1 | 子ども | 大きくてカラフルなひらがなを見たい | 文字が見やすく楽しい |
| 2 | 子ども | ひらがなの音を聞きたい | 読み方を耳で覚えたい |
| 3 | 子ども | 文字をなぞって書く練習がしたい | 書き順を体で覚えたい |
| 4 | 子ども | クイズに正解したらほめてもらいたい | やる気が続く |
| 5 | 子ども | 好きなキャラクターと一緒に練習したい | 楽しくなる |
| 6 | 保護者 | 子どもの進捗を確認したい | どこまで覚えたか把握したい |
| 7 | 保護者 | 練習する文字の範囲を設定したい | 段階的に学習させたい |

## 3. 機能要件

### 3.1 学習モード（みる・きく）
- ひらがな一覧（五十音表）をタイル表示
- 文字をタップすると大きく表示 + 読み上げ音声を再生
- 文字に関連するイラスト（例：「あ」→ アリのイラスト）を表示
- アニメーションで書き順を表示

### 3.2 なぞり書きモード（かく）
- キャンバス上に薄く表示されたひらがなをなぞる
- 書き順ガイド（番号付き矢印）を表示
- 正しくなぞれたら音と光でポジティブフィードバック
- 失敗しても怒らない（やさしいフィードバックのみ）

### 3.3 クイズモード（あてる）
- **音声クイズ**: 読み上げた音に合う文字を選ぶ（4択）
- **文字クイズ**: 表示された文字の読み方を選ぶ（4択）
- **イラストクイズ**: イラストに合う文字を選ぶ（4択）
- 正解でアニメーション・音・褒め言葉
- 不正解でも「もう一回！」と優しく促す

### 3.4 ご褒美・モチベーション機能
- スター（★）コレクション：文字をクリアするたびに獲得
- スタンプカード：一定数クリアでシールがもらえる演出
- キャラクターの成長：練習量に応じてキャラが育つ
- 効果音・BGM（かわいい・テンポよい）

### 3.5 保護者向け機能（設定画面）
- 学習進捗の確認（習得済み文字の可視化）
- 練習する文字セットの選択（あ行のみ、など）
- 音量調整・BGMオン/オフ
- 子どもロック（誤操作防止のため設定変更には保護者操作が必要）

## 4. 非機能要件

### 4.1 ユーザビリティ
- タップ対象は最小 **60×60px** 以上（指でタップしやすいサイズ）
- テキスト指示は最小限。アイコンとイラストで操作を伝える
- 読み込み時間は **2秒以内**（子どもは待てない）
- 文字・ボタンは高コントラストで視認性を確保

### 4.2 対応環境
- **プラットフォーム**: Webブラウザ（スマートフォン・タブレット優先）
- **対応デバイス**: iOS Safari、Android Chrome、iPad
- **画面サイズ**: 320px〜1280px のレスポンシブ対応
- **オフライン**: 基本機能はオフラインでも動作（PWA対応）

### 4.3 アクセシビリティ
- 音声読み上げに対応（すべての文字・ボタンに音声あり）
- 色覚多様性を考慮したカラーパレット

### 4.4 安全性
- 広告なし
- 外部リンクなし
- 個人情報の収集なし（保護者の設定データはローカル保存）

## 5. 画面構成

```
ホーム画面
├── 学習モード（みる・きく）
│   └── 五十音一覧 → 文字詳細（音声・イラスト・書き順アニメ）
├── なぞり書きモード（かく）
│   └── 文字選択 → なぞり書きキャンバス
├── クイズモード（あてる）
│   └── クイズ種類選択 → 4択クイズ → 結果・フィードバック
├── コレクション（ほしのへや）
│   └── 獲得スター・スタンプ・キャラクター確認
└── 設定（保護者向け）
    └── 進捗・文字セット設定・音量
```

## 6. 技術スタック（候補）

| カテゴリ | 技術 | 理由 |
|---------|------|------|
| フロントエンド | React + TypeScript | コンポーネント管理・型安全 |
| スタイリング | Tailwind CSS | 素早くかわいいUIを構築 |
| なぞり書き | Canvas API / fabric.js | 手書き入力の制御 |
| 音声 | Web Speech API + 音声ファイル | 読み上げ・効果音 |
| 状態管理 | Zustand | シンプルで軽量 |
| PWA | Vite PWA Plugin | オフライン対応 |
| デプロイ | Vercel / GitHub Pages | 無料・簡単 |

## 7. ひらがな学習コンテンツ

### 7.1 文字セット
- **基本46文字**: あいうえお〜ん（清音）
- **拡張（将来対応）**: 濁音（が行など）、半濁音（ぱ行）、拗音（きゃなど）

### 7.2 各文字のコンテンツ
各ひらがなに対して以下を用意する：

| 項目 | 内容 | 例（あ） |
|------|------|---------|
| 読み | ひらがなの読み | あ |
| 例単語 | その文字が頭文字の単語 | アリ、アメ |
| イラスト | 例単語に対応する絵 | アリの絵 |
| 書き順データ | SVGパスまたは座標列 | 3画の順序 |
| 音声ファイル | 読み上げ音声 | "あ" の発音 |

| フェーズ | 内容 | 優先度 |
|---------|------|--------|
| Phase 1 | 学習モード（みる・きく）+ 五十音一覧 | 最高 |

| Phase 3 | なぞり書きモード | 中 |
| Phase 4 | ご褒美・コレクション機能 | 中 |
| Phase 5 | 保護者設定・進捗確認 | 低 |
| Phase 6 | PWA対応・パフォーマンス最適化 | 低 |

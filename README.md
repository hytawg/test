# ScreenStudio

Mac に最適化した画面収録・編集デスクトップアプリ。無音デモ動画（X へのポスト用など）の作成に特化しています。

## 機能

- **画面収録** — ディスプレイ全体またはウィンドウ（タブ）単位でキャプチャ
- **カメラオーバーレイ** — 円形・角丸・四角のピクチャーインピクチャー
- **キャンバスカスタマイズ** — 34種のグラジェント背景、macOS壁紙風プリセット、単色、ブラー、カスタム画像
- **カーソルオーバーレイ** — 録画中のマウス位置を白い円で可視化（フォーカスログが有効な場合）
- **録画後エディタ**
  - **Trim** — 開始・終了点の調整 ＋ 途中区間のカット（Cut）
  - **Zoom** — 開始〜終了・倍率・フォーカス点を管理。エッジドラッグでリサイズ可能
  - **Text** — テキスト注釈（位置・表示時間・スタイル）
  - **Speed** — 区間ごとに再生速度を変更。エッジドラッグでリサイズ可能
  - **Auto Zoom** — 録画中のカーソル軌跡をもとに自動カメラワーク
- **動画ファイルインポート** — 外部の動画ファイル（MP4・WebM・MOV・MKVなど）をエディタで開いて編集
- **録画履歴** — エクスポートした録画を一覧表示し、いつでも再編集
- **エクスポート** — MP4 / WebM / GIF、品質・FPS 選択、Downloads 自動保存

## 必要環境

- macOS
- Node.js 18 以上

## 使い方

### 1. ソースを選ぶ（Source）

左サイドバーの **Source** アイコンをクリックし、録画するディスプレイまたはウィンドウを選択します。

- **Displays** — 画面全体をキャプチャ
- **Windows** — 特定アプリのウィンドウのみキャプチャ。ブラウザを選ぶとヘッダー・アドレスバー・ブックマークバーを自動でクロップします

### 2. 見た目を整える（Canvas）

**Canvas** パネルで背景・パディング・角丸・アスペクト比を調整します。

| 項目 | 説明 |
|---|---|
| Padding | 録画映像の周囲の余白サイズ |
| Corner Radius | 映像の角の丸み |
| Shadow | ドロップシャドウの有無・強度 |
| Background | グラジェント（34種）/ macOS壁紙 / 単色 / ブラー / カスタム画像 / なし |
| Aspect Ratio | 16:9 / 4:3 / 1:1 / 9:16 / Fill |

### 3. カメラを設定する（Camera）※任意

**Camera** パネルでウェブカメラのオーバーレイを有効にできます。形（円・角丸・四角）・位置・サイズを調整できます。

### 4. 録画する

画面下部の赤い **Record** ボタンをクリックすると 3 秒カウントダウン後に録画開始。

| ボタン | 動作 |
|---|---|
| Record | 録画開始 |
| ⏸ | 一時停止 |
| ▶ | 再開 |
| Stop & Edit | 録画停止してエディタへ |
| ✕ | 録画キャンセル |

### 5. 編集する

録画停止後、自動的にエディタ画面に遷移します。左ツールバーでツールを切り替えます。

#### ✂ Trim（トリム・カット）

タイムライン上の紫色のハンドルをドラッグして開始・終了点を調整します。

**途中区間のカット（新機能）**

タイムライン下部の赤い **Cut** レーンをドラッグすると、その区間がエクスポート時にスキップされます。

| 操作 | 動作 |
|---|---|
| ドラッグ | カット区間を作成 |
| エッジをドラッグ | 区間の開始・終了点を調整 |
| 区間をドラッグ | カット位置を移動 |
| ✕ ボタン | カット区間を削除 |

カット区間はメイントラックにストライプ柄で表示されます。右パネルの Trim ツールにカット一覧と合計削除時間が表示されます。

#### 🔍 Zoom（ズーム）

ズーム領域は「開始時間〜終了時間・倍率・フォーカス点」を1セットで管理します。

**追加方法（3種類）**

| 操作 | 動作 |
|---|---|
| タイムラインをドラッグ | ドラッグした範囲でズーム領域を作成 |
| タイムラインを2回クリック | 1クリック目で開始点、2クリック目で終了点を指定 |
| プレビューをクリック | クリックした位置・時刻から 2 秒間のズーム領域を作成 |

**編集**

- タイムライン上のバーの**左右エッジをドラッグ**して区間を調整（Text と同じ操作感）
- バー中央をドラッグして位置を移動
- Zoom パネルで倍率・フォーカス点・イージングを調整

#### T Text（テキスト注釈）

テキストツールに切り替えてプレビュー上をクリックするとテキストを配置できます。タイムライン上の注釈バーのエッジをドラッグして表示時間を変更できます。

| 設定項目 | 内容 |
|---|---|
| テキスト内容 | 自由入力 |
| 表示時間 | タイムラインのバーをドラッグまたはパネルで調整 |
| フォントサイズ | スライダー |
| 文字色 / 背景色 | カラーピッカー |
| 太字 / 揃え | ボタン切替 |

#### ⚡ Speed（速度調整）

**追加方法**

| 操作 | 動作 |
|---|---|
| タイムラインをドラッグ | ドラッグした範囲で速度区間を作成 |
| タイムラインを2回クリック | 1クリック目で開始点、2クリック目で終了点を指定 |

作成後はバーの**左右エッジをドラッグ**して区間を調整、バー中央をドラッグで移動。Speed パネルで 0.25×〜4.0× の倍率を設定できます。

#### ⚡ Auto Zoom（自動ズーム）

録画中のカーソル軌跡を記録し、エディタ上部の **Auto Zoom** ボタンで有効にするとカーソルを追うカメラワークが自動で適用されます（フォーカスログが有効な場合のみ表示）。カーソル位置は白い円のオーバーレイとして映像上に表示されます。

### 6. 動画ファイルをインポート・再編集する（Files）

左サイドバーの **Files** アイコンをクリックします。

**動画ファイルのインポート**

| 操作 | 動作 |
|---|---|
| ドロップゾーンにドラッグ&ドロップ | 動画ファイルをエディタで開く |
|「Browse File…」ボタン | ファイル選択ダイアログを開く |

対応形式: MP4 / WebM / MOV / MKV / AVI / M4V

**録画履歴からの再編集**

エクスポートのたびにファイルが履歴に自動追加されます。

| 操作 | 動作 |
|---|---|
| ↩ ボタン | ファイルをディスクから読み込んで再編集 |
| ✕ ボタン（ホバー時） | 履歴から削除（ファイル本体は削除しない） |

---

### 7. エクスポートする

エディタ右パネル下部の **Export** ボタンをクリック。カット・トリム・ズーム・速度変更・テキストがすべて反映されたファイルを生成します。

| 設定 | 選択肢 |
|---|---|
| Format | MP4 / WebM / GIF |
| Quality | High (8Mbps) / Medium (4Mbps) / Low (2Mbps) |
| FPS | 24 / 30 / 60 |
| 保存先 | Downloads 自動保存 / ダイアログで選択 |

## ビルド（配布用 .app の生成）

```bash
npm run build
npm run build:mac
```

`out/` フォルダに `ScreenStudio.app` が生成されます。

## 初回起動時の注意

macOS の画面収録権限が必要です。

`システム設定 → プライバシーとセキュリティ → 画面収録` でアプリを許可してください。
